# 旧原型归档

这里保存的是 `小智 / Xiaozhi` 在正式切换到 `Next.js + TypeScript + React` 之前的一版静态原型。

## 归档目的

- 保留早期内容结构、教学文案和交互表达，方便后续逐个迁移。
- 将旧站点资产移出仓库根目录，避免与生产工程代码、路由和测试混淆。
- 明确这些文件默认不参与当前构建、Lint、TypeScript 检查和 Playwright 验证。

## 目录说明

- `assets/`: 旧版样式、内容数据和基础引擎实现。
- `demos/`: 旧版演示器源码与共享插件基类。
- `modules/`: 旧版模块页和单元页静态 HTML。
- `subjects/`: 旧版学科入口页。
- `tests/`: 仅面向旧原型的历史测试资产。
- `index.html`: 旧版首页入口。

## 使用原则

- 当前生产开发默认只改根目录下的正式工程，不直接在这里继续迭代功能。
- 当某个知识点需要迁移时，可从这里提取教学表达、参数逻辑和可视化思路，再按现行架构重写。
- 如果后续确认某一批旧资产完全没有参考价值，再做二次清理，而不是在归档阶段直接删除。
