# 小智项目概览

## 1. 项目是什么

小智（Xiaozhi）是一个面向高中阶段的知识可视化学习项目。

它关注的不是“题目答案”，而是：

- 一个知识点为什么会这样
- 它内部有哪些结构关系
- 参数变化后规律如何变化
- 学生怎样从“记结论”走向“真正理解”

因此，小智采用的不是传统知识库式页面，而是：

- 结构化正文
- 图形化知识表达
- 可交互 demo
- 学习路径串联

## 2. 面向用户

- 高中学生
- 需要用更直观方式讲概念的家长或老师
- 希望通过图形与结构理解，而不是只靠刷题记忆的学习者

## 3. 当前内容范围

当前正式站点已覆盖：

- 数学
- 物理
- 化学
- 英语

已完成结构接入：

- `4` 个学科
- `17` 个模块
- `72` 个知识点

## 4. 产品原则

### 4.1 先建立结构理解

每个知识点都尽量回答：

- 它的核心结构是什么
- 变量之间如何联动
- 学生最容易混淆的地方是什么

### 4.2 图形化不是装饰，而是解释工具

图形、示意板、参数交互的目标不是好看，而是：

- 把抽象对象具象化
- 把变化过程可视化
- 把不同概念的关系放到同一视图里

### 4.3 手机端和电脑端都要可用

重构过程中已明确要求双端兼容，因此：

- 页面布局需要移动端可读
- demo 不能依赖只适合大屏的旧原型壳子
- 窄屏下也要保持内容结构清晰

### 4.4 迁移不是复刻

旧原型只作为迁移输入，不再作为正式运行入口。

当前项目采用：

- 迁移同时重构
- 允许适度重组
- 不改变学科主线

## 5. 技术形态

当前正式站点基于：

- Next.js
- React
- TypeScript
- App Router
- Vitest
- Playwright

同时已经形成三层结构：

1. 内容层：学科、模块、知识点、学习路径、MDX 正文
2. 页面层：学科页、模块页、知识点页
3. 交互层：demo 注册表、统一 demo shell、图形/结构化渲染能力
